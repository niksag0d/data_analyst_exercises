## Task 4 – Time Series & KPI Analysis (COVID‑19)

This task involves KPI tracking and time series analysis using OWID COVID‑19 data.

### **Objectives**

* Load OWID dataset
* Filter selected countries
* Calculate rolling averages
* Plot time series
* Detect KPI alerts
* Summarize trends

### **Data to Download**

OWID Dataset: [https://docs.owid.io/projects/covid/en/latest/dataset.html](https://docs.owid.io/projects/covid/en/latest/dataset.html)

### **Instructions**

**KPIs to Calculate (per country)**

* 7‑day avg new cases
* 7‑day avg deaths
* cumulative cases per 100k

**Visualizations**

* 7‑day avg new cases per country
* cumulative cases per 100k

**Short Analysis**

* Compare peak timings
* Which country recovers fastest?

**Bonus**

* Alert if 7‑day avg > threshold
* Print dashboard summary

### **Key Questions**

* When did each country peak?
* How do trends differ?
* Is the situation improving or worsening?
